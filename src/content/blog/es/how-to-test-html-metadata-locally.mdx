Usualmente para probar los [metatags](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/meta) necesitamos hacer un deploy, podemos ahorra tiempo probando localmente asi:

## [Primero](#first)

Debemos tener nuestra aplicacion corriendo en algun puerto desde nuestra terminal

```bash
-> npm start

> portfolio@3.0.0 start
> astro developer
> Local http://localhost:3000
```

## [Segundo](#second)

Necesitamos tener [npm](https://nodejs.org/en/download) instalado y ejecutamos la instrucción:

```bash
npx localtunnel --port 3000
```
Si es la primera vez que utilizas [localtunnel](https://theboroer.github.io/localtunnel-www/), veras una pregunta en la terminal, presionas **Y** y obtienes una **url** similar a esta:

```bash
-> npx localtunnel --port

> your url is: https://strong-meals-prove-181-142-210-248.loca.lt
```

## [Tercero](#third)

Nos dirijimos a [metatags.io](https://metatags.io/) y en la parte superior de la página agregas la **url** que nos generó [localtunnel](https://theboroer.github.io/localtunnel-www/)

![](/assets/test-metatags/metatag-search-bar.png)

Con esta herramientas puedes validar y editar cada metatag

![](/assets/test-metatags/metatags-preview.png)

En la parte izquierda, hay opciones que permiten previsualizar como se vería en **Facebook**, **Google** o **Twitter**, entre otros.

## [Por último](#last)
Puedes copiar la configuración utilizando en botón de **Generate** y llevarlo a tu web

![](/assets/test-metatags/metatags-generated.png)
