Usually, to test [metatags](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/meta), we need to deploy. We can save time by testing locally like this:

## [First](#first)

We need to have our application running on some port from our terminal.

```bash
-> npm start

> portfolio@3.0.0 start
> astro developer
> Local http://localhost:3000
```

## [Second](#second)

We need to have [npm](https://nodejs.org/en/download) installed and execute the command:

```bash
npx localtunnel --port 3000
```

If it's the first time you use [localtunnel](https://theboroer.github.io/localtunnel-www/), you will see a question in the terminal. Press **Y** and you will get a **URL** similar to this:

```bash
-> npx localtunnel --port

> your url is: https://strong-meals-prove-181-142-210-248.loca.lt
```

## [Third](#third)

We go to [metatags.io](https://metatags.io/) and in the top of the page we add the **URL** generated by [localtunnel](https://theboroer.github.io/localtunnel-www/).

![](/assets/test-metatags/metatag-search-bar.png)

With this tool, you can validate and edit each metatag.

![](/assets/test-metatags/metatags-preview.png)

On the left side, there are options that allow you to preview how it would look on **Facebook**, **Google**, or **Twitter**, among others.

![](/assets/test-metatags/metatags-generated.png)
